# Wanderlust Project
Wanderlust is a full-stack travel listing web application where users can explore, create, edit, and review travel destinations.
This project is built using the MERN backend stack with authentication, image upload, and session management.


Tech Stack

Backend: Node.js, Express.js
Database: MongoDB Atlas
Templating Engine: EJS
Authentication: Passport.js
Session Store: connect-mongo
Image Upload: Cloudinary + Multer
Deployment: Render

Features

🔐 User Authentication (Register/Login/Logout)
🏡 Create, Edit, Delete Listings
🖼 Image Upload with Cloudinary
⭐ Add & Delete Reviews
🛡 Authorization (Only owner can edit/delete)
💾 Session stored in MongoDB
📱 Responsive UI
